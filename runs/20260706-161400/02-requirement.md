# Requirement

## Original request
Build me a page that says "Hello" with a big red button

## Summary
Create a single web page displaying the text "Hello" along with one large red button.

## Acceptance criteria
- The page loads and displays the text "Hello" somewhere visible on it.
- The page contains a button that is visibly large (noticeably bigger than default button size).
- The button is red in color.
- Only one button is present on the page.
- The page renders correctly with no console errors.
