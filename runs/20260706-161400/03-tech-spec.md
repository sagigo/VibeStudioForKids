# Technical specification

## App type
Single static HTML page.
No interactivity, routing, or state is required by the acceptance criteria, so a single page is sufficient.

## Stack
Plain HTML with inline CSS.
The button needs no behavior (no click handler specified), so vanilla HTML/CSS covers every acceptance criterion without a build step or framework.

## Storage
None.
There is no data to persist or retrieve; content is static text and markup.

## Deployment target
Static file(s) served via GitHub Pages.
