# Tasks

## Development
1. Create the single static HTML file with the required content and markup structure.
2. Add inline CSS to style the page and the (non-interactive) button per the acceptance criteria.
3. Set up the repository/folder structure for GitHub Pages deployment (e.g. index.html at the correct path) and enable Pages.

## Testing
1. Open the HTML file in a browser and visually verify all required content and the button render correctly.
2. Verify the deployed GitHub Pages URL loads the page correctly.
